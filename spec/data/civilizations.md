# Civilization Data Reference

All 45 civilizations (base game through Mountain Royals DLC). Unique units are trained at the Castle (standard form) and can be upgraded to Elite form via Castle research. Chronicle DLC civs (Achaemenids, Athenians, Jurchens, Khitans, etc.) are out of scope for this clone.

## CivEntry TypeScript interface

```typescript
export interface CivEntry {
  id: string;
  name: string;
  uniqueUnitIds: string[];    // Castle-trainable unique units (both base and elite forms)
  bonuses: CivBonus[];        // Passive bonuses applied at game start or on age-up
}

export interface CivBonus {
  description: string;        // Human-readable
  minAge?: UnitAge;           // null = active from game start
  effects: TechEffect[];      // Applied via TechSystem
}
```
